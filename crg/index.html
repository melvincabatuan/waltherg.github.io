What is GraTeLPy?

Software for the parameter-free analysis of chemical reaction networks. This user-friendly software allows you to analyze if a reaction network of interest has the capability to show multistability, pattern formation, or oscillations. GraTeLPy is easy to use, entirely free, and can be deployed to compute clusters -- making the analysis of large reaction networks feasible.

Overview

With GraTeLPy you can rapidly decide if your chemical reaction network has the potential to show multistability, patterns, or oscillations.

GraTeLPy is a new software package that determines if your chemical reaction network (CRN) meets necessary conditions for multistability, pattern formation, or oscillations. This method represents your CRN as a bipartite directed graph and determines if this graph contains certain elements, "critical fragments", that are necessary for the aforementioned phenomena -- GraTeLPy can tell you with certainty when your CRN does not have the potential to display these phenomena. If your CRN does have this potential, GraTeLPy can guide estimation of parameter values so that these phenomena are observed.

Applications

At present, GraTeLPy is applicable to all chemical reaction networks with mass-action kinetics. Read on for an overview of published analyses.
